# Deployment Workflow

Verify production build.

Check environment variables.

Check custom domain.

Verify HTTPS.

Verify redirects.

Verify cache headers.

Verify robots.txt.

Verify sitemap.xml.

Verify 404 handling.

Confirm deployment succeeded.

Generate deployment notes.
