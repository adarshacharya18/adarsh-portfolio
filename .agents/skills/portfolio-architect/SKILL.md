# Portfolio Architect

## Goal

Maintain a production-quality software engineering portfolio.

The portfolio should always reflect these personality traits:

- Calm
- Professional
- Hardworking
- Productive
- Adaptive
- Minimal

Never create flashy designs.

---

## Tech Stack

React

Vite

TypeScript

Tailwind CSS

Framer Motion

React Icons

Cloudflare Pages

---

## Design Principles

Prefer whitespace.

Low contrast.

Subtle shadows.

Smooth animations.

Apple / Linear / Vercel inspiration.

Dark mode first-class.

Responsive on every screen.

---

## Architecture Rules

Never hardcode portfolio data.

Use JSON or Markdown.

Keep components reusable.

Avoid duplication.

Prefer composition.

Use strict TypeScript.

No inline CSS.

No unnecessary dependencies.

## Coding Standards

Single responsibility.

Reusable components.

Semantic HTML.

Accessibility first.

Performance first.

Lazy load large sections.

Use Framer Motion conservatively.

Never sacrifice performance for animation.
