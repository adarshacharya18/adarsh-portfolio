# Review Checklist

Before considering a task complete:

✔ Responsive

✔ Dark Mode

✔ Accessible

✔ No console errors

✔ ESLint passes

✔ Types pass

✔ Mobile friendly

✔ Animation smooth

✔ Lighthouse friendly

✔ Components reusable

If any check fails,

fix it before responding.
